# Typography

Typography plays a crucial role in defining the visual identity of your application and conveying information to users.
The UI Kit provides a set of typography-related CSS variables that you can use to ensure consistent and visually
appealing text throughout your design system.

## Font Sizes

The UI Kit defines a range of font sizes that can be used to create harmonious text hierarchies. Here are some of the
font size variables available:

-   **Font Size 75:** `--0xf-font-size-75`
-   **Font Size 87:** `--0xf-font-size-87`
-   **Font Size 100:** `--0xf-font-size-100`
-   **Font Size 125:** `--0xf-font-size-125`
-   **Font Size 150:** `--0xf-font-size-150`
-   **Font Size 200:** `--0xf-font-size-200`
-   **Font Size 250:** `--0xf-font-size-250`
-   **Font Size 300:** `--0xf-font-size-300`
-   **Font Size 400:** `--0xf-font-size-400`

## Font Weights

Font weight can be used to add emphasis and hierarchy to your text. The UI Kit provides the following font weight
variables:

-   **Font Weight Light:** `--0xf-font-weight-light`
-   **Font Weight Medium:** `--0xf-font-weight-medium`
-   **Font Weight Bold:** `--0xf-font-weight-bold`

## Font Styles

Font style can affect the appearance of your text, such as italicizing. The UI Kit offers the following font style
variables:

-   **Font Style Normal:** `--0xf-font-style-normal`
-   **Font Style Italic:** `--0xf-font-style-italic`

## Body Typography

The UI Kit also defines variables specifically for body typography. These variables can be used to maintain a consistent
text style for your application's content:

-   **Font Family:** `--0xf-body-font-family`
-   **Font Size:** `--0xf-body-font-size`
-   **Line Height:** `--0xf-body-line-height`
-   **Font Weight:** `--0xf-body-font-weight`
-   **Font Style:** `--0xf-body-font-style`
-   **Text Rendering:** `--0xf-body-text-rendering`

To use these typography-related variables, apply them to your stylesheet rules and elements to ensure a harmonious and
visually appealing text presentation throughout your application.
