# Global Styles

Global styles define consistent visual elements that impact the overall look and feel of your application. The UI Kit
defines a set of global style variables that can be used to maintain uniformity across different components and sections.

## Scroll Bars

Customizing scroll bars can enhance the visual appeal of your application's user interface. The UI Kit provides
variables to control the appearance of scroll bars:

-   **Scrollbar Width:** `--0xf-global-scrollbar-width`
-   **Scrollbar Track Color:** `--0xf-global-scrollbar-track-color`
-   **Scrollbar Thumb Color:** `--0xf-global-scrollbar-thumb-color`
-   **Scrollbar Thumb Border Radius:** `--0xf-global-scrollbar-thumb-border-radius`

Use these variables to adjust the width and color of scroll bars as well as the border radius of the scroll thumb.

## Border Radius

Border radius is a fundamental design element that affects the curvature of corners in UI components. The UI Kit defines
the following border radius variables:

-   **Small Radius:** `--0xf-global-border-radius-s`
-   **Medium Radius:** `--0xf-global-border-radius-m`
-   **Large Radius:** `--0xf-global-border-radius-l`
-   **Extra Large Radius:** `--0xf-global-border-radius-xl`

These variables allow you to consistently apply border radius values across your components, ensuring a cohesive and
visually pleasing design.

By utilizing these global style variables, you can maintain a consistent design language throughout your application
and easily adapt and customize the visual elements as needed.
